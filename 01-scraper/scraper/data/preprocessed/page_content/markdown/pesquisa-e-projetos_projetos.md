| Análise de risco: uma metodologia voltada para segurança | |
| --- | --- |
| **Resumo**: Este projeto apresenta uma proposta de estudo e desenvolvimento de uma metodologia de análise de risco em sistemas c omputacionais. A análise incorpora a avaliação do risco e seu gerenciamento, isto é, ela combina processos sistemáticos para a identificação do risco e suas conseqüências, e como lidar ou tratar estes riscos. Através desta pesquisa, pretendemos modelar uma metodologia para análise de risco, a partir de padrões já conhecidos, voltada estritamente à segurança da informação, adequada de acordo com os padrões da norma ISO 17799\. A metodologia de análise de risco deve avaliar se um sistema computacional atende aos requisitos básicos de segurança de acordo com o seu propósito, e enquadrado na norma ISO 17799, fornecendo ao final um documento com a análise completa dos riscos, probabilidades de ocorrências, custos de uma possível ocorrência, contra\-medidas e os custos das contra\-medidas a serem adotadas a fim de evitar tais ocorrências.**Equipe**: Adriano Mauro Cansian (coordenador); Dioraci Corrêa Jr. (mestrando), Allan de Brito Mendez Calderon (aluno de graduação)**Recursos**: | |

| Sistemas de segurança para redes de computadores | |
| --- | --- |
| **Resumo**: Visando identificar, codificar, analisar e entender as técnicas utilizadas pelos atacantes em sistemas de computadores, foi desenvolvido o conceito de honeypot. O honeypot é um dispositivo computacional interligado em uma rede, e se caracteriza por um sistema proposital e especialmente configurado para ser alvo de uma tentativa de ataque. Desta forma, pode\-se gerar informações necessárias à compreensão dos mecanismos utilizados para a realização e, ocasionalmente, sucesso da intrusão. Como conseqüência do entendimento do ataque, é possível desenvolver sistemas de defesa. O conceito de honeynet refere\-se a uma rede controlada ligando diversos honeypots à Internet. Este projeto de pesquisa pretende implementar uma honeynet segura para aplicação em pesquisas de segurança, desenvolver um sistema capaz de identificar a ocorrência de uma intrusão e bloquear qualquer ataque originado a partir da honeynet. Para tal, pretende\-se adaptar uma honeynet prototipada já existente em nosso laboratório, e desenvolver um sistema de controle baseado em um conjunto de bibliotecas de software para implementação e aplicação de sistemas de firewall encontrado no kernel do Linux nas versões 2\.4\.X e superiores. Além disso, pretende\-se utilizar intensivamente a honeynet aprimorada como elemento para a coleta de assinatura de ataques de intrusão, bem como elemento de obtenção de artefatos de risco, tais como viruses e worms, para serem utilizados em rotinas de análise e resposta a incidentes, e em outros procedimentos e rotinas de segurança de computadores e redes.**Equipe**: Adriano Mauro Cansian (coordenador), Jorge Luiz Corrêa (mestrando), André Proto (aluno de graduação), Isabela Liane de Oliveira (aluna de graduação), Allan de Brito Mendez Calderon (aluno de graduação), Leandro Arabi Alexandre (aluno de graduação)**Recursos**: | |

| Detecção precoce de problemas de perfuração de poços de petróleo por meio da análise de cascalhos | |
| --- | --- |
| **Resumo**: Os cascalhos gerados durante a perfuração de poços de petróleo são uma importante fonte de informação para o diagnóstico de ocorrências de anormalidades. Os cascalhos, em geral, apresentam formas e tamanhos semelhantes quando a perfuração está sendo executada em operação normal. Mudanças no formato ou no tamanho dos cascalhos podem indicar a ocorrência de anormalidades, como o desmoronamento do poço. Os processos atuais de análise de cascalhos são manuais e, por isso, podem demorar muito para detectar eventos inesperados. Há, portanto, a necessidade de se desenvolverem técnicas para detecção automática e precoce de problemas durante a perfuração de poços de petróleo. O principal objetivo desse projeto de pesquisa é estudar e avaliar técnicas de processamento de imagens e de classificação para a análise dos cascalhos produzidos durante o processo de perfuração de poços de petróleo, visando a detecção automática e precoce de problemas.**Equipe**: Aparecido Nilceu Marana (colaborador), Evandro de Castro Carmo (aluno de graduação), Ivan Rizzo Guilherme (colaborador), Kazuo Miura (coordenador externo)**Recursos**: | |

 



| Desenvolvimento de dispositivo anagliptográfico para inclusão digital de deficientes visuais | |
| --- | --- |
| **Resumo**: O projeto visa a construção de um dispositivo eletromecânico, reconfigurável em tempo real, capaz de exibir todos os diferentes sinais do alfabeto Braille em uma matriz de pontos que se elevam e abaixam em uma superfície de referência. | Segmentação dentária utilizando a Transformada Imagem\-Floresta | |
| --- | --- |
| **Resumo**: A evolução da tecnologia e o grande volume de ocorrências que necessitam de investigações realizadas por especialistas forenses têm estimulado o desenvolvimento de sistemas automáticos para identificação humana forense. A identificação forense post\-mortem (PM) não pode, obviamente, ser realizada por meio de características comportamentais do indivíduo e, em determinadas circunstâncias, nem mesmo por meio da maioria de suas características físicas, uma vez que estas podem ser destruídas durante ou após a morte. Características dentárias de um indivíduo são consideradas únicas e as mais apropriadas para identificação PM, uma vez que elas persistem mesmo após a morte. A identificação de indivíduos através de registros dentários é aceita pelas autoridades legais, pois é um método que fornece resultados precisos e confiáveis para identificação positiva. Apesar do progresso das últimas duas décadas, o problema de segmentação de imagens continua sendo um dos mais difíceis desafios em análise de imagens. Os métodos de segmentação são normalmente complexos, caros computacionalmente, dependentes de aplicação e freqüentemente exigem interação humana. Métodos de segmentação interativa de imagens que reduzem a intervenção do usuário a escolhas de poucos pixels vêm sendo pesquisados. Uma possibilidade promissora desse tipo de abordagem usa a Transformada Imagem\-Floresta (IFT \- Image Foresting Transform), que reduz o problema de processamento de imagem ao cálculo de uma floresta de caminhos de custo mínimo no grafo derivado da imagem. O objetivo deste projeto é implementar e avaliar algoritmos para segmentação automática ou semi\-automática de imagens de radiografias de arcadas dentárias, baseadas na Transformada Imagem\-Floresta, visando a detecção precisa dos contornos dos dentes com mínima intervenção humana. Pretende\-se, posteriormente, incorporar o algoritmo mais eficaz ao sistema automático para identificação de dentes sendo desenvolvido no Laboratório de Computação de Alto Desempenho (LCAD), do Departamento de Computação, da Faculdade de Ciências, Unesp, campus de Bauru.**Equipe**: Aparecido Nilceu Marana (coordenador), Elizabeth Bonsaglia Barboza (aluna de graduação), Juan Rogélio Falguera (ex\-aluno mestrado)**Recursos**: Bolsa IC FAPESP | |
